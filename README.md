# 🤖 Client Outreach Agent

[![AI](https://img.shields.io/badge/AI-Gemini-blueviolet)](https://deepmind.google/technologies/gemini/)
[![Framework](https://img.shields.io/badge/Framework-Next.js%2015-black)](https://nextjs.org/)
[![Database](https://img.shields.io/badge/DB-MongoDB-green)](https://www.mongodb.com/)

> **Find and email local businesses without websites.**
> Automate your outreach with personalized AI-generated emails.

---

## 🚀 Key Features
- **Local Business Discovery**: Find businesses via Google Maps.
- **Contact Extraction**: Scrape emails and social media from business sites.
- **AI Outreach**: Generate personalized emails using Gemini.
- **Bulk Sending**: Automate delivery via Nodemailer.

---

## 🚦 Getting Started
1. Clone the repo.
2. Setup `.env.local` with `MONGODB_URI`, `GEMINI_API_KEY`, and SMTP settings.
3. Run `npm install && npm run dev`.
